Étape 2 : Équipages

La deuxième étape a pour but de recenser tous les équipages participant à la régate.

Principe général

Vous allez devoir remplir un grand tableau : chaque ligne correspond à un équipage différent

Juste après la clôture des inscriptions, ajoutez chaque équipage (nom, et bateaux si classement en temps compensé).

Ajouter et supprimer un équipage

À tout moment, vous pouvez rajouter un nouvel équipage en cliquant sur le bouton Ajouter un équipage situé au-dessus du tableau.

Pour supprimer un équipage, il suffit de cliquer sur la croix grise située à gauche du nom de l’équipage correspondant.

Nom de l’équipage

Vous devez nécessairement donner un nom à chaque équipage. C’est ce nom qui apparaîtra sur le classement final.

Il peut être judicieux d’y inclure le numéro de bateau, afin de simplifier ensuite la rentrée des résultats des manches (c’est souvent le numéro des bateaux qui est noté à l’arrivée).

Bateaux

Si vous avez choisi un classement en temps compensé, vous devez aussi renseigner le bon bateau. Pour cela, deux solutions :

  • Si vous connaissez déjà le « code » correspondant, il suffit de le rentrer dans le champ de la colonne Code. Si ce code est correct, le nom complet du bateau doit apparaître dans la colonne Type de bateau.
  • Sinon, utilisez le champ de la colonne Type de Bateau pour rechercher le bateau : dès qu'une lettre est tappée, une liste déroulante apparaît, vous proposant toutes les possibilités correspondant à ce que vous avez déjà tappé. Même s’il ne reste plus qu’un bateau dans la liste, pensez bien à cliquer sur son nom pour le sélectionner. Le code correspondant doit apparaître à gauche : il peut être intéressant de le mémoriser pour une prochaine fois…

Remarque : vous pouvez également rentrer un cœfficient dans le champ de la colonne code. Ce cœfficient sera alors utilisé pour calculer le temps compensé de l’équipage correspondant. Cela peut être utile par exemple si un bateau n’est pas présent dans la liste déroulante mais que connaissez le cœfficient à utiliser.
