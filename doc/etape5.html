Étape 5 : Résultats

Après avoir renseigné toutes les informations nécessaires au calcul du classement (étapes 1 à 4), il ne reste plus qu’à cliquer sur Résultats en bas à gauche pour effectivement calculer le classement.

Erreurs courantes

Si les informations renseignées aux étapes 1 à 4 sont incomplètes, le calcul du classement est stoppé et un message d’information est affiché afin que vous puissiez corriger le problème. Une fois ce problème corrigé, vous pourrez relancer le calcul du classement.

Erreurs courantes

  • un équipage n’a pas de nom : vous devez absolument nommer tous les équipages, c’est ce qui les identifie sur le classement final !
  • un type de bateau n’existe pas (classement temps compensé uniquement): un des équipages n’a pas de type bateau renseigné, ou ce type de bateau est incorrect. Pour corriger ce problème, assurez vous d’avoir sélectionné un bateau dans la liste déroulant de la colonne Type de bateau (la colonne code se rempli alors automatiquement) ou d’avoir renseigné un code correct (la colonne Type de bateau se remplit alors automatiquement). Attention : suivant les ratings utilisés, il se peut que le type de bateau voulu ne soit pas disponible !
  • aucune manche n’a encore été ajouté : les résultats ne peuvent être calculés que si au moins une manche a été courue !

Affichage des résultats

Après avoir cliqué sur Résultats, le classement général de la régate doit apparaître. Attention : cela peut prendre quelques secondes s’il y a beaucoup d’équipages et de manches (une barre de progression est affichée en bas à droite de la fenêtre).

Pour voir les classements de chaque manche, utilisez la liste déroulante située au-dessus du classement.

Imprimer un classement

Il n’est pour l’instant pas possible d’imprimer un classement directement depuis DériVoile Calc’.

Cependant, il vous suffit d’exporter ce classement en PDF (cf. ci-dessous), de l’ouvrir avec votre visionneuse PDF (probablement Acrobat Reader) et de l’imprimer !

Diffuser un classement

Tout d’abord, choisissez le classement que vous voulez diffuser dans la liste déroulante. Ensuite, cliquez sur Exporter en HTML… ou Exporter en PDF… dans le menu Classement.

Exporter en PDF

Le format PDF est parfait pour partager un classement par email par exemple, ou tout simplement pour ensuite pouvoir l’imprimer. Tous les ordinateurs disposent d’une visionneuse PDF : tout le monde pourra donc consulter le classement.

Exporter en HTML

Le format HTML est parfait pour partager un classement sur internet (via le site web du club organisateur par exemple).
